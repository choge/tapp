TA Protein Predictor is a tool for prediction of Tail-anchored proteins, proteins with a single transmembrane domain and lacking signal peptides. This predictor and predicts the input sequences are TA proteins or not based on Hidden Markov model.

Input your sequences below in fasta format.

The predicted result will be shown in a new page.

The number of queries is limited to 50. If you want to predict more than the limitation, please use the client tool from Github. https://github.com/davecao/tappm_cli

The data sets used for training the model can be downloaded:

TA set SP set MP set NO set
